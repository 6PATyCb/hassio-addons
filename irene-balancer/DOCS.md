## Как установить в HA?

Вам нужно подключить [мой репозиторий](https://github.com/6PATyCb/hassio-addons) в HA и из него уже будет доступна установка аддона.

## Как это работает?
Запускает nginx с upstream, где есть есть основной сервер и резервный.

В настройках укажите ссылку на основную Ирину через переменную `TARGET_1` и резервную Ирину через `TARGET_2`.
Теперь в плагинах [STT](https://github.com/6PATyCb/irene_stt) и [TTS](https://github.com/6PATyCb/irene_tts) ссылайтесь на балансировщик вместо конкретной Ирины (балансировщик обычно запускается на порту 5013).

Доступна страница мониторинга https://homeassistant.local:5013/status.html где можно увидеть состояние балансировки.

## Какие есть известные проблемы и решения?

Не проверялось, когда HA работает под HTTPS.

Не проверялось, когда Ирина работает под HTTP.

## Как запустить в докере?
Для запуска локальном докере выкачайте проект и соберите Dockerfile командой:
```
docker build --build-arg BUILD_FROM=ghcr.io/hassio-addons/base:19.0.0 -t irene-balancer .
```

Для запуска под виндой контейнера в локальном докере выполните команду:
```
docker run --rm -it ^
  --name irene-balancer ^
  -p 5013:5013 ^
  -e LISTEN_PORT="5013" ^
  -e TARGET_1="https://192.168.133.252:5003" ^
  -e TARGET_2="https://192.168.144.112:5003" ^
  -e DOMAIN="test.local" ^
  irene-balancer
```
