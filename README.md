# hassio-addons
Репозиторий моих аддонов для HA

- [Irene-Voice-Assistant](https://github.com/6PATyCb/hassio-addons/tree/main/irene-ha-addon) - Обертка для запуска [Ирины](https://github.com/janvarev/Irene-Voice-Assistant) внутри HA

## Установка

Вы должны использовать [Hass.io](https://www.home-assistant.io/hassio/) - supervised [Home Assistant](https://www.home-assistant.io/) версию для поддержки работы с аддонами.

Чтобы подключить репозиторий аддонов, вы должны зайти в HA в Настройки -> Приложения -> Установить приложение (кнопка снизу справа) -> ... (кнопка сверху справа) -> Репозитории -> Добавить (кнопка снизу справа) 

Вписать в открывшееся поле адрес:
```
https://github.com/6PATyCb/hassio-addons
```

Теперь репозиторий подключен и вы можете искать приложение в разделе Настройки -> Приложения -> Установить приложение (кнопка снизу справа)
